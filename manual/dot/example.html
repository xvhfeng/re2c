Below is an example of generating a picture of DFA that accepts any UTF-8 code point (utf8_any.re):

/*!re2c
    *   {}
    [^] {}
*/

Generate and render:

$ re2c -D -8 utf8_any.re | dot -Tpng -o utf8_any.png
